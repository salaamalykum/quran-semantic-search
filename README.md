## 一、项目背景与初衷

### 中文穆斯林社区面临的现实困境

《古兰经》是真主降示给全人类的最后启示，其原文以古典阿拉伯语写成，字字珠玑，层叠的修辞与音韵之美是任何翻译都无法完全再现的。然而，对于全球超过千万的能看懂中文的穆斯林而言，能够直接研读阿拉伯原文的人毕竟是少数。绝大多数信徒，尤其是中国内地、马来西亚、印度尼西亚、新加坡等地的穆斯林，以及遍布欧美各国的以汉语为母语的穆斯林移民，主要依靠中文译本来理解真主的话语。

这便带来了一个长期存在却鲜少被系统性解决的问题：**中文译本数量众多，质量参差不齐，而一般信众往往只接触到其中一个版本**。每一位译者，无论学识多么渊博，都是时代的产物，都有其语言风格的局限与侧重。当你手中只有一本马坚译本时，你未必能感受到王静斋阿訇在某个关键词汇上的不同选择背后所隐含的神学考量；当你只读过仝道章的字对字直译时，你也许会错过马仲刚白话版中更贴近现代语境的诠释。

单一译本，就像从一扇窗户看庭院，而并行五个译本，才是推开了五扇不同朝向的窗，让光从各个角度射入。

### 为什么需要一个专门的检索引擎？

传统的经书阅读方式，虽然神圣而庄重，但在实际查找特定经文方面存在明显局限：

**其一，查阅效率低下。** 如果你记得某段经文大致的意思，却不记得它在哪一章哪一节，翻阅整本经书可能耗费大量时间。对于学习阶段的新穆斯林，这种困难尤为突出。

**其二，版本比较困难。** 要将五个译本的同一节经文并排对照，传统方式意味着你需要同时翻开五本厚重的经书，对齐同一章节，这在日常学习中几乎不具备可操作性。

**其三，关键词检索缺失。** 如果你想查阅《古兰经》中所有涉及"施舍"、"忍耐"、"公正"或"孤儿"的经文，传统阅读方式无法帮助你做到这一点。而主题式学习，恰恰是深化信仰认知、准备讲道（呼图白）或进行学术研究最有效的方法之一。

**其四，移动互联网时代的需求。** 今天的穆斯林，无论是在公交车上、午休时间，还是在礼拜前的等待片刻，都希望能够随手查阅经文。一个针对手机端优化的轻量检索引擎，填补了这一空白。

### 我们的承诺

本项目承诺：

- **永久免费**，不收取任何费用，不投放任何广告。
- **开源透明**，所有代码公开，任何人都可以审查、复制和改进。
- **忠实原典**，所有经文内容均来自公认的权威译本，不作任何删改。
- **持续维护**，随着社区反馈不断修正错误、优化体验。

## 二、五大权威中文译本详介

本项目精心收录了以下五位在中国伊斯兰翻译史上地位卓著的学者的经典译本。了解每位译者的学术背景与翻译风格，将帮助你更有智慧地使用并行对照功能。

### 1. 马坚译本（Ma Jian）

马坚教授（1906—1978），云南沙甸人，是中国现代伊斯兰学术史上最具影响力的学者之一。他早年赴埃及爱资哈尔大学深造，接受了系统的阿拉伯语和伊斯兰经典教育，归国后长期执教于北京大学东方语言系。

马坚译本以其**古朴典雅的文言与白话过渡风格**著称，译文庄重严谨，在忠实原文语义的同时兼顾了汉语的表达习惯。该译本获得沙特阿拉伯法赫德国王古兰经印制厂的认可，作为官方汉语版本大量印制发行，是目前全球华语穆斯林中流传最广、使用最普遍的版本，也是许多清真寺和伊斯兰学校的首选教材。

**适合人群：** 具备一定汉语文学素养的成年穆斯林；希望获得权威标准参照的学者；准备引用经文进行演讲或撰文的人士。

### 2. 马金鹏译本（Ma Jinpeng）

马金鹏先生的译本以**详尽的注释和通俗化的诠释**见长。他在翻译过程中不仅关注经文本身的字面意思，还广泛引用传统伊斯兰注经学（塔夫斯尔）的资源，对许多阿拉伯语中难以直译的修辞表达（巴拉盖）给出了清晰的脚注说明。

对于那些希望深入了解某一节经文背景——包括降示时的历史情境（舍班·努祖勒）——的读者，马金鹏译本提供了极为丰富的辅助信息。这使得它不仅是一个翻译，更近似于一部简明的中文古兰经注释手册。

**适合人群：** 有志于系统学习伊斯兰知识的求学者；希望了解经文降示背景的信众；从事阿拉伯语与汉语对照研究的学者。

### 3. 王静斋译本（Wang Jingzhai）

王静斋阿訇（1879—1949），天津人，是中国近代伊斯兰四大阿訇之一，也是中国穆斯林文化复兴运动（伊赫瓦尼运动）的重要推动者。他一生致力于伊斯兰经典的汉译工作，先后完成了《古兰经》的甲、乙、丙三个不同版本的翻译，每一版都是在前一版的基础上进一步修订完善的。

王静斋译本具有鲜明的**时代烙印**，其语言风格带有清代至民国时期白话文运动过渡阶段的特征，对于研究中国伊斯兰汉文化（汉克塔布文化）的学者而言，这是极具价值的第一手历史文献。通过他的译本，我们不仅能读到《古兰经》的中文诠释，还能感受到一百年前中国穆斯林学者在特定历史文化语境中理解和表达信仰的方式。

**适合人群：** 对中国伊斯兰历史与文化有研究兴趣的学者；希望了解汉克塔布文化传统的人士；对不同历史时期中文宗教文本有比较研究需求的研究者。

### 4. 仝道章译本（Tong Daozhang）

仝道章先生的译本采取了严格的**字对字直译（Word-for-Word）**方法论，将他的译本与其他版本对照阅读，能够帮助读者判断哪些中文表达是原文直接对应，哪些是译者的语义延伸。

**适合人群：** 正在学习阿拉伯语的穆斯林学子；希望理解原文句法结构的研究者；进行古兰经语言学研究的专业人士。

### 5. 马仲刚译本（Ma Zhonggang）

马仲刚先生的译本在五个版本中最具**现代白话风格**，语言简洁流畅，贴近当代汉语读者的阅读习惯。他在翻译时充分考虑到当代穆斯林，尤其是青少年和新皈依者的阅读需求，避免了文言词汇和过于复杂的句式结构。

对于刚刚接触伊斯兰的新穆斯林，或是希望以轻松方式开始阅读《古兰经》的年轻一代，马仲刚译本是极佳的入门读本。其通俗易懂的语言风格，也使其成为父母向子女介绍经文内容的理想工具。

**适合人群：** 新皈依穆斯林；青少年穆斯林；希望用当代语言理解信仰的普通信众。

## 三、如何使用本检索引擎

### 基本操作流程

本检索引擎的设计原则是**零学习成本**，任何会使用智能手机搜索功能的人都能立刻上手。

**第一步：进入检索页面**

根据您使用的设备选择合适的入口：
- 电脑用户请访问：[https://salaamalykum.com/cn/qurancn/pc/](https://salaamalykum.com/cn/qurancn/pc/)
- 手机用户请访问：[https://salaamalykum.com/cn/qurancn/mobile/](https://salaamalykum.com/cn/qurancn/mobile/)

**第二步：输入关键词**

在页面顶部的搜索框中输入您想查找的关键词。您可以输入：
- 经文中的具体词语，例如："天课"、"斋戒"、"忍耐"、"公正"
- 伊斯兰概念的汉语名称，例如："圣战"、"礼拜"、"朝觐"
- 人名或地名，例如："易卜拉欣"、"麦加"、"麦地那"
- 章名，例如："开端章"、"黄章"、"古莱什章"

**第三步：查看并行对照结果**

系统将在同一个屏幕内，将五位译者对包含该关键词的所有经文节的翻译并排列出。您可以：
- 横向比较同一节经文在不同译本中的表达差异
- 点击特定章节跳转查看上下文
- 在五个译本之间自由切换，专注于某一个版本阅读

### 使用场景举例

**场景一：礼拜前的诵读准备**

你明天要在清真寺带领会众诵读一段经文，但你希望先理解其中某个词汇的深层含义。你可以在检索引擎中输入该词汇，同时看到五位学者各自的翻译选择，这将给你提供更立体的理解基础。

**场景二：回答非穆斯林朋友的问题**

一位非穆斯林朋友问你，《古兰经》中关于"宽恕"的教导是什么。你可以迅速检索"宽恕"，找出相关经文，并借助多个译本的对照，为朋友提供更准确、更有说服力的解答。

**场景三：青少年伊斯兰教育**

家长或经文班教师希望给学生讲解某一主题，例如"感恩"或"孝顺父母"。通过主题关键词检索，可以快速汇集所有相关经文，形成完整的主题教学材料。

**场景四：个人信仰研修**

在每日功课之外，你希望对某一苏拉（章节）进行深入研读。通过查阅五个译本的并行呈现，你可以发现不同学者在某些关键词上的翻译分歧，从而引发更深层的思考，这本身就是一种非正式的塔夫斯尔（注经）学习。

## 四、本项目的学术与数据价值

### 填补中文伊斯兰数字人文研究的空白

在阿拉伯语、英语、波斯语、乌尔都语等语言中，古兰经的数字化资源已经相当成熟，有多个成熟的在线平台提供多译本对照功能。然而，在中文领域，这类资源长期处于分散、低质量、缺乏系统整合的状态。本项目是目前已知的、第一个将五个主要权威中文译本完整数字化并提供全文关键词检索的开源平台。

### 对中文伊斯兰学术研究的贡献

本项目整理的数据集，对以下研究领域具有直接的学术价值：

**中文宗教翻译学研究：** 通过比较同一阿拉伯原文在不同时代、不同译者手中产生的汉语文本差异，研究者可以系统分析影响宗教文本翻译策略的语言、文化和神学因素。

**汉语伊斯兰文化史研究：** 五个译本横跨的时间段涵盖了从清末到当代的汉语伊斯兰文化发展历程，为相关历史研究提供了珍贵的第一手文献。

**计算语言学与自然语言处理：** 高质量的平行多版本宗教文本数据集，是训练和评估中文语言模型在宗教领域语义理解能力的理想资源，尤其是在词义消歧（Word Sense Disambiguation）和同义表达识别等任务上。

**伊斯兰教育学研究：** 不同译本在词汇选择上的差异，为研究伊斯兰概念如何在汉语文化语境中被理解、接受和传播提供了丰富的实证材料。

### 开放数据集说明

本项目的所有经文数据以结构化 JSON 格式存储，并提供以下标准化数据集文件供研究人员使用：

- `quran_rag_dataset.jsonl`：五译本并行对照数据集，JSON Lines 格式，每行为一节经文的五译本对照记录
- `quran_rag_alpaca.jsonl`：转化为 Alpaca 指令跟随格式的数据集，适合用于语言模型的指令微调实验

数据集的使用须遵守本项目的开源许可协议，并在任何学术发表中适当注明来源。

## 五、本地部署指南（个人电脑离线版）

伊斯兰的知识不应受制于网络连接是否稳定。本项目采用**无服务器纯静态架构**，这意味着你完全可以将整个系统下载到本地，在没有网络的情况下照常使用。这对于网络条件不稳定的地区——无论是中国农村地区的清真寺、非洲的穆斯林社区，还是网络受限的特殊环境——都具有重要的现实意义。

### 系统要求

- 任何一台能运行现代浏览器（Chrome、Firefox、Safari、Edge）的电脑或手机
- 约 50MB 的本地存储空间
- **不需要**安装任何编程环境（不需要 Python、Node.js、Docker 等）
- **不需要**网络连接（下载完成后）

### 下载步骤

**方法一：通过 ZIP 压缩包下载（推荐普通用户）**

1. 访问本项目的 GitHub 仓库页面
2. 点击页面右上角的 **"Download"** 按钮，选择 **"Download ZIP"**
3. 等待下载完成（文件约 30-50MB）
4. 在你的电脑上解压缩该 ZIP 文件，例如解压到桌面的 `Quran_Search` 文件夹

**方法二：通过 Git 克隆（推荐有技术基础的用户）**

如果你的电脑已经安装了 Git，可以在终端或命令提示符中运行：

```bash
git clone https://github.com/your-username/quran-semantic-search.git
```

### 启动本地检索引擎

**电脑端：**

1. 打开解压后的文件夹
2. 进入 `pc` 子文件夹
3. 找到 `index.html` 文件
4. 双击打开，系统默认浏览器将自动启动并显示检索界面

**手机端（通过手机访问已下载到电脑的文件）：**

如果你希望在局域网内通过手机访问本地部署的系统，可以在电脑端用任意简单的 HTTP 服务器提供文件。如果你安装了 Python，可以在项目根目录运行：

```bash
# Python 3
python3 -m http.server 8080
```

然后在手机浏览器中访问 `http://你电脑的局域网IP地址:8080/mobile/index.html`。

### 验证安装成功

启动后，在搜索框中输入"麦加"或"斋月"，如果界面正确显示了五个译本的对照结果，说明安装成功。

## 六、服务器云端部署指南

对于希望为社区建立在线伊斯兰图书馆的阿訇、伊斯兰学校的技术负责人，或公益组织的 IT 志愿者，本节提供完整的服务器部署指南。

### 适用场景

- 清真寺或伊斯兰中心希望为会众提供专属的经文检索服务
- 伊斯兰学校希望搭建内网学习平台
- 公益机构希望以自己的域名运营这一服务
- 个人开发者希望在自己的服务器上运行并进行二次开发

### 环境要求

- Linux 服务器（Ubuntu 20.04 LTS 或以上推荐）
- Nginx 或 Apache web 服务器
- 可选：SSL 证书（强烈建议用于公网部署）
- 磁盘空间：约 100MB
- 内存：最低 512MB（本系统为纯静态，对服务器资源要求极低）

### 部署步骤

**步骤一：克隆代码库到服务器**

通过 SSH 登录到你的服务器，然后执行：

```bash
# 进入你希望存放网站文件的目录
cd /var/www/

# 克隆代码库
git clone https://github.com/your-username/quran-semantic-search.git

# 进入项目目录
cd quran-semantic-search

# 确认文件结构完整
ls -la
```

你应该能看到 `pc/`、`mobile/`、`data/` 等目录。

**步骤二：配置 Nginx**

创建一个新的 Nginx 站点配置文件：

```bash
sudo nano /etc/nginx/sites-available/quran-search
```

输入以下配置内容（根据你的实际情况修改域名和路径）：

```nginx
server {
    listen 80;
    server_name your-domain.com www.your-domain.com;

    # 网站根目录（指向克隆的项目文件夹）
    root /var/www/quran-semantic-search;
    index index.html;

    # 启用 GZIP 压缩，显著提升加载速度
    # 五译本数据量较大，开启压缩可减少约 60% 的传输体积
    gzip on;
    gzip_vary on;
    gzip_min_length 1024;
    gzip_types
        text/plain
        text/css
        text/javascript
        application/javascript
        application/json
        text/xml
        application/xml;

    # 静态资源缓存策略（JSON 数据文件较大，建议长期缓存）
    location ~* \.(json|js|css)$ {
        expires 7d;
        add_header Cache-Control "public, no-transform";
    }

    # 前端路由支持
    location / {
        try_files $uri $uri/ /index.html;
    }

    # 安全头部（推荐）
    add_header X-Frame-Options "SAMEORIGIN";
    add_header X-Content-Type-Options "nosniff";
}
```

**步骤三：启用站点配置并重启 Nginx**

```bash
# 创建软链接启用站点
sudo ln -s /etc/nginx/sites-available/quran-search /etc/nginx/sites-enabled/

# 测试配置文件语法是否正确
sudo nginx -t

# 重载 Nginx 使配置生效
sudo systemctl reload nginx
```

**步骤四：配置 HTTPS（强烈推荐）**

如果你的服务器有公网域名，强烈建议使用 Let's Encrypt 配置免费的 HTTPS 证书，保护用户的检索请求：

```bash
# 安装 Certbot
sudo apt install certbot python3-certbot-nginx

# 自动获取并配置 SSL 证书
sudo certbot --nginx -d your-domain.com -d www.your-domain.com

# 证书会自动续期，无需手动维护
```

**步骤五：设置文件权限**

确保 Nginx 进程有权限读取项目文件：

```bash
sudo chown -R www-data:www-data /var/www/quran-semantic-search
sudo chmod -R 755 /var/www/quran-semantic-search
```

### 验证部署成功

在浏览器中访问你的域名，在搜索框中输入任意关键词，如果正常显示五译本对照结果，说明部署成功。

### 使用 Apache 替代 Nginx

如果你的服务器使用 Apache，可以在项目根目录创建 `.htaccess` 文件实现相同效果：

```apache
Options -Indexes
RewriteEngine On
RewriteBase /
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ /index.html [L,QSA]

# 启用压缩
<IfModule mod_deflate.c>
    AddOutputFilterByType DEFLATE text/html text/css application/javascript application/json
</IfModule>

# 设置缓存
<IfModule mod_expires.c>
    ExpiresActive On
    ExpiresByType application/json "access plus 7 days"
    ExpiresByType application/javascript "access plus 7 days"
</IfModule>
```

## 八、常见问题解答

**Q：这个工具可以替代正式的伊斯兰教育吗？**

不可以，也不应该。本工具是一个辅助研读的数字工具，它可以帮助你快速检索和比较译文，但它无法替代在合格学者指导下系统学习古兰经注释学（塔夫斯尔）、阿拉伯语语法（纳赫乌）以及伊斯兰法学（菲格赫）的重要性。我们鼓励每一位使用本工具的穆斯林，在条件允许时继续寻求合格阿訇或伊斯兰学者的指导。

**Q：哪个译本是"最准确"的？**

这个问题没有简单的答案。每个译本都有其学术价值和适用场景。《古兰经》原文的阿拉伯语有着无限的语义深度，任何单一译本都只能是对其中某些层面的近似。这正是我们提供五个译本并列的原因——通过比较，读者可以感受到不同学者对同一原文的不同理解，从而获得更接近真相的整体认知。

**Q：我可以将这个工具用于清真寺的伊斯兰教育吗？**

完全可以，这正是本项目的重要使用场景之一。本项目是完全免费的公益项目，任何清真寺、伊斯兰学校、穆斯林社区组织都可以自由使用，包括在自己的服务器上部署私有版本。

**Q：网站支持哪些浏览器？**

本系统使用现代标准 Web 技术构建，支持所有主流浏览器的近两年版本，包括：Chrome、Firefox、Safari、Edge 以及它们的移动端版本。对于较旧的浏览器（如 Internet Explorer），我们不提供技术支持。

**Q：我发现了一处经文翻译错误，如何反馈？**

感谢你的认真阅读和对数据质量的关注。请通过以下方式之一提交反馈：
- 在 GitHub 仓库页面提交 Issue，详细说明章节编号、节数、涉及的译本，以及你发现的问题
- 发送邮件至 [bropeace@protonmail.com](mailto:bropeace@protonmail.com)，标题注明"经文勘误反馈"

**Q：我可以为这个项目贡献代码或内容吗？**

非常欢迎。请参阅下一节"贡献与参与"的详细说明。

**Q：本项目是否有手机 App 版本？**

目前没有独立的 App，但 Mobile 端检索页面经过专门的响应式设计优化，在智能手机浏览器中有流畅的使用体验。你可以将网页添加到手机主屏幕，以类似 App 的方式使用。
